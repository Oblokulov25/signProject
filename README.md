# React Authentication System (Sign In, Sign Up, Forgot Password)

Ushbu loyiha zamonaviy va xavfsiz autentifikatsiya tizimini o'z ichiga oladi. Loyihada foydalanuvchini ro'yxatdan o'tkazish, tizimga kirish va **EmailJS** orqali parolni tiklash funksiyalari to'liq ishlaydi.

## ✨ Xususiyatlari
- **Sign Up:** Yangi foydalanuvchilarni ro'yxatdan o'tkazish.
- **Sign In:** Tizimga kirish oynasi.
- **Forgot Password:** Parolni unutganlar uchun EmailJS orqali 4 xonali tasdiqlash kodi yuborish tizimi.
- **Responsive Design:** Barcha qurilmalar (mobil, planshet, desktop) uchun moslashtirilgan.
- **Animations:** Tailwind CSS va framer-motion orqali yaratilgan chiroyli vizual effektlar.
